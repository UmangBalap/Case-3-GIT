# Format Text
teks bisa di tulis tebal, miring, dan strikeline dengan cara 
1. **tebal** pake 2 bintang ** tebal **
2. *miring* pake 1 bintang * miring *
3. ~~strikeline~~ pake tanda samping no 1 ~~ strikeline ~~
